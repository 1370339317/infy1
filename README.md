# InfinityHookPro
InfinityHookPro Win7 -> Win10 latest

最近来了兴致,拿起了InfinityHook,原始的版本有点小问题,后来改了支持Win7到Win10 1909的

参考了网上大佬发出来的各种版本,最终出了这份代码,理论上支持Win7到最新版本的Win10了

代码里面有很详细的注释,不懂的地方可以结合博客
